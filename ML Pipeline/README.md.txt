# Titanic Survival Prediction - ML Pipeline Project

## 📌 Project Overview
This is a beginner-friendly Machine Learning project demonstrating how to build a complete **ML Pipeline** using Scikit-learn.

The project predicts whether a passenger survived the Titanic disaster using features like age, sex, fare, class, etc.

---

## 🚀 Features of This Project
- Data preprocessing using `ColumnTransformer`
- Handling missing values using `SimpleImputer`
- Encoding categorical features using `OneHotEncoder`
- Scaling numerical features using `StandardScaler`
- Model training using `LogisticRegression`
- Evaluation using accuracy score and classification report
- Saving trained pipeline using `joblib`
